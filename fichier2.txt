WebRTC s'assure que le téléphone Android et le PC sont capables de se connecter via le réseau Wi-Fi local.
Lorsque l'utilisateur sélectionne un fichier à envoyer, il envoie ce fichier via WebRTC directement au PC.
Pourquoi Git te demande de faire un commit ?

    Git te demande de valider les résolutions de conflits après chaque étape du rebase, mais au lieu de faire un commit classique, il te demande de continuer le rebase avec git rebase --continue.
    Avantages de cette solution

    Pas de serveur local sur le PC : L'utilisateur n'a pas besoin de configurer un serveur sur son PC.
